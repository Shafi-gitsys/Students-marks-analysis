# Student Marks Analysis System Using Python

## Description
This project analyzes student marks using Python. It calculates total, average, highest and lowest marks, assigns grades, and shows data in tabular and graphical format.

## Technologies Used
- Python
- Google Colab
- Pandas
- Matplotlib

## Features
- Total and average calculation
- Grade allocation
- Data visualization
- Table display
